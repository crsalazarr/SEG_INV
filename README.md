SEG_INV - Sistema de Inventario y Compras
-Descripción

SEG_INV es una aplicación web desarrollada en ASP.NET Core MVC que permite gestionar inventarios, compras y usuarios dentro de una organización. El sistema facilita el control de productos, el seguimiento de entradas y la administración de roles.

-Funcionalidades

-Autenticación y usuarios
Inicio de sesión
Gestión de usuarios
Control de roles (Administrador, Bodega, Inventario, etc.)

-Inventario
Registro de productos
Consulta de stock
Actualización de cantidades

-Bodega
Gestión de bodegas
Organización de productos

-Compras
Registro de compras
Ingreso de productos al inventario
Seguimiento de operaciones

-Paneles por rol
Vistas diferentes según el tipo de usuario
Interfaces como:
Administrador
Inventario
Bodega
Gerente

-Tecnologías
C#
ASP.NET Core MVC
MySQL
Bootstrap
HTML / CSS

-Arquitectura
El sistema está basado en el patrón MVC:

Controllers → Lógica de control
Models → Datos y conexión a BD
Views → Interfaz de usuario
